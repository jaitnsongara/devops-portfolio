# Kubernetes Demonstrations

## Overview
Production-ready Kubernetes configurations showcasing microservices deployment, scaling, and management.

## Projects Included

1. **Microservices Application** - Multi-container deployment with service mesh
2. **Auto-Scaling Configuration** - HPA and VPA implementations
3. **StatefulSet with Persistent Storage** - Database deployment patterns
4. **Helm Charts** - Package management and templating

## Skills Demonstrated
- Pod, Deployment, and Service configurations
- ConfigMaps and Secrets management
- Ingress and load balancing
- Horizontal Pod Autoscaling
- StatefulSets and persistent volumes
- Helm chart development
