# Terraform Demonstrations

## Overview
Infrastructure as Code examples demonstrating AWS resource provisioning, state management, and best practices.

## Projects Included

1. **AWS Multi-Tier Architecture** - VPC, EC2, RDS, S3
2. **Auto-Scaling Web Application** - ALB, ASG, CloudWatch
3. **Secure Infrastructure** - IAM, Security Groups, KMS

## Skills Demonstrated
- Resource provisioning and management
- State management and remote backends
- Module creation and reusability
- Variable management and workspaces
- Output values and data sources
