# Docker Demonstrations

## Overview
Containerization examples showcasing multi-stage builds, optimization, and best practices.

## Projects Included

1. **Multi-Stage Build** - Optimized Node.js application
2. **Docker Compose Stack** - Full application stack with networking
3. **Security Hardened Images** - Non-root users, minimal base images

## Skills Demonstrated
- Dockerfile optimization and multi-stage builds
- Docker Compose orchestration
- Volume management and networking
- Security best practices
- Image size optimization
